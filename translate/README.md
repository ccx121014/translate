# 自动翻译插件

一个自动将网页内容翻译成中文的浏览器插件。

## 功能特性

- 自动检测网页语言并翻译成中文
- 支持多种源语言和目标语言
- 可开关翻译功能
- 翻译结果缓存，提高性能
- 支持长文本翻译
- 自动重试机制，提高翻译成功率

## 安装方法

1. 下载或克隆此仓库到本地
2. 打开Chrome浏览器，进入`chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择插件所在的文件夹
6. 插件安装完成，在浏览器右上角可以看到插件图标

## 使用方法

1. 点击浏览器右上角的插件图标
2. 切换"启用翻译"开关来开启或关闭翻译功能
3. 可以选择源语言和目标语言
4. 当翻译功能开启时，访问的网页会自动翻译成选择的目标语言

## 注意事项

1. 本插件使用Google Translate API进行翻译
2. 由于使用的是免费API，翻译速度可能受到限制
3. 某些网站可能会阻止翻译功能

## 开发说明

### 主要文件说明

- `manifest.json`：定义插件的基本信息、权限和资源
- `background.js`：处理插件的后台逻辑和翻译API调用
- `content.js`：在网页上执行翻译功能
- `popup.html`：插件的弹出窗口界面
- `popup.js`：处理弹出窗口的用户交互
